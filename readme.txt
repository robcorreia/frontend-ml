Para rodar o projeto é necessário os seguintes passos:

back-end
- acessar a pasta do projeto backend
- rodar um yarn ou npm install para instalar as dependências do projeto
- rodar o comando yarn dev ou npm run dev

front-end
- acessar a pasta do projeto frontend
- rodar um yarn ou npm install para instalar as dependências do projeto
- rodar o comando yarn start ou npm start
